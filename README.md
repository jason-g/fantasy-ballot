# fantasy-ballot

simple app for tracking user picks for ballot questions

Still major work in progress!

# Install:
- npm install
- npm run clean
- npm run build

# Run the server:
- npm run start

# Test the endpoints:

- REST:  http://localhost:3000/explorer/
- GraphQL: http://localhost:3001/graphql
- openAPI: http://127.0.0.1:3000/openapi.json
